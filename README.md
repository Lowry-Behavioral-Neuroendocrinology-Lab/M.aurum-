QIIME commands were saved in Maurum_Microbiome_Jupyter_SAS_Updated_042526ipynb. QIIME version "qiime2-2024.10" was used.

Statistical analysis and figure making was done using R in Maurum_Microbiome_Visualizations.Rmd. An html printed version of this file is available as High_fat_microbiome_analysis.html. Also, other measurements (body weights, rt-PCR of hippocampus choroid plexus, and linear regressions were run for statistical analysis and M.aurum_bodyweights-rt-PCR-linearregression.Rmd and html can be downloaded. 

This folder can be downloaded and used as an R project in R studio using the Maurum_Microbiome.Rproj file to start an R project
